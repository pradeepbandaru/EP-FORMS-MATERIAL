Important Information:-

  • 1. Set templateOptions.validator to specify the validation for input field.
  • 2. Set templateOptions.validationMsg to specify the validation message for input field.
  • 3. Note: We can use custom validation. For this we use 'Custom' in .validator field And also specify regex property for perticuler custom validation.

Options that are available for this validations are given below:

  • Phone:For phone number validation. ex: 111-111-1111
  • ZIP:For zip code validation. ex: 55555-4444
  • Email:For email address validation. ex: test@domail.com
  • url:For web url validation starts with 'https://'. ex: https://test.com
  • ip-address:For ip address validation. ex: 255.255.255.259
